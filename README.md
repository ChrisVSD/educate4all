## 🎯 Αποστολή
Να δημιουργήσουμε μια πλήρως δωρεάν, ανοιχτού κώδικα πλατφόρμα εκπαίδευσης με τεχνητή νοημοσύνη για όλους.

## 🚀 Τι Προσφέρουμε
- 🤖 **AI Βοηθός Εκπαίδευσης** - Δωρεάν βοηθός ΤΝ για μαθητές και εκπαιδευτικούς
- 📚 **Πλούσιο Υλικό** - Μαθήματα και ασκήσεις για όλα τα επίπεδα
- 🌍 **Πρόσβαση για Όλους** - Χωρίς συνδρομές, χωρίς κρυφά κόστη
- 🔓 **Ανοιχτού Κώδικα** - Ολόκληρο το project είναι ανοιχτό για συνεισφορές

## 🛠️ Τεχνολογίες
- Python / FastAPI
- React/Next.js  
- Hugging Face Transformers
- PostgreSQL

## 👥 Πώς Μπορείς να Βοηθήσεις
Ψάχνουμε:
- 👨‍💻 Προγραμματιστές (Python, JavaScript)
- 🎓 Εκπαιδευτικούς
- 🎨 Designers
- 🌐 Μεταφραστές
