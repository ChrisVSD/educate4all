## 📖 Περιγραφή

Το **educate4all** είναι μια καινοτόμος εκπαιδευτική πλατφόρμα που χρησιμοποιεί Τεχνητή Νοημοσύνη για να προσφέρει προσωποποιημένη μάθηση. Σχεδιασμένη με στόχο τη δωρεάν πρόσβαση στην εκπαίδευση για όλους, η πλατφόρμα μας προσφέρει έξυπνους AI βοηθούς, δομημένα μαθήματα και διαδραστικές εμπειρίες μάθησης.

> 🎯 **Στόχος μας**: Να κάνουμε την ποιοτική εκπαίδευση προσβάσιμη σε όλους, χωρίς οικονομικά εμπόδια.

## ✨ Χαρακτηριστικά

### 🤖 Πολλαπλοί AI Βοηθοί
- **🧠 Πραγματικός AI** - Για σύνθετα προβλήματα και λεπτομερείς αναλύσεις
- **🎯 Έξυπνος AI** - Για βαθιά κατανόηση και εξηγήσεις βήμα-βήμα  
- **😊 Απλός AI** - Για γρήγορες ερωτήσεις και απλές εξηγήσεις

### 📚 Ολοκληρωμένα Μαθήματα
- **📐 Μαθηματικά** - Άλγεβρα, Γεωμετρία, Λογισμός
- **🔬 Φυσική** - Μηχανική, Ηλεκτρομαγνητισμός, Σύγχρονη Φυσική
- **🧪 Χημεία** - Οργανική, Ανόργανη, Αντιδράσεις
- **🧬 Βιολογία** - Κυτταρική, Γενετική, Ανατομία
- **🌍 Ιστορία & Γεωγραφία** - Αρχαία Ελλάδα, Σύγχρονος Κόσμος
- **💼 Οικονομικά** - Μακροοικονομία, Αγορές, Χρηματοοικονομικά
- **🤖 Τεχνολογία** - Προγραμματισμός, Ρομποτική, AI
